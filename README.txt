To install dependencies use the command "npm install" from ternimal within the project directory.

The project report file named "Project_report" is in the Project folder.

Important Instructions:
1) Please run the mongod in your device.
2) Before running the server, you must install all the 
   dependencies in the Project folder where the movieServer.js file 
3) Run the database-initializer.js file.
4) Run the server using command "node movieServer.js"
